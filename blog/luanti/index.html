Getting Started with Luanti

About

#

Luanti is an open source voxel game engine that lends itself particularly well to buidling things together with friends. Naturally, I run a server.

Here you will find my notes on running MineGame on Luanti, for friends and family. Except where otherwise noted, my server is typically Debian and my usual client is my SteamDeck.

These are my notes on my setup. As always, this is ostensibly for your benefit.

More honestly, these notes are so I can re-build my setup later after I inevitably mess it up while tinkering.
