Remote Access

#

Under Preferences

  • Change Password
  • Under Interfaces - Enable SSH and VNC

On a Windows computer:

choco install vnc-viewer

VNC remote host will be raspberrypi.local. User is pi. Password is whatever you set in the step above.

Overclock

#

Watch your speed with one of these commands:

cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_cur_freq
vcgencmd measure_clock arm
watch -n 1 vcgencmd measure_clock arm

To edit the overclock setting:

sudo vi /boot/config.txt

# uncomment to overclock the arm. 700 MHz is the default.
over_voltage=6
arm_freq=2000

Reboot to apply changes.

sudo reboot
